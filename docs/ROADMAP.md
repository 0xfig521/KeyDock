# KeyDock Roadmap

## v0.1 MVP

- Secret CRUD.
- Workspace CRUD.
- Workspace variable mapping.
- CLI `run`, `copy`, and `export env`.
- Basic Tauri desktop UI.
- Local encrypted SQLite storage.
- Audit log without plaintext.

## v1.0

- Polished desktop UX.
- Import/export flows with explicit confirmation.
- Better onboarding and provider templates.
- Stronger unlock/lock lifecycle.
- CLI integration tests and packaging.

## v1.5

- Project binding model surfaced in UI.
- Directory-aware workspace selection in CLI.
- Optional shell helper for explicit activation.
- Manual provider health checks.

## v2.0

- Cloud sync.
- Team sharing.
- Secret rotation.
- RBAC.
- Provider and launcher plugin systems.
