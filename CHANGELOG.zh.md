# 更新日志

## 0.6.0 - 2026-06-08

### 新功能
- 用可复用、可组合、可激活的预设替代旧的工作区模型。
- 新增密钥字段与预设条目，将加密密钥字段直接映射为环境变量。
- 新增预设 UI、预设组合、激活/停用、预览，以及 `keydock run` 单命令注入流程。
- 围绕本地 API Key 保险库、可复用环境变量预设和 AI Agent 安全注入刷新官网 SEO 与文案。

### 重构
- 移除旧 Key 和工作区相关的前后端代码路径，并将审计/环境变量命名统一到预设。
- 从文档、应用文案和官网文案中移除产品层面的 Workspace 术语。

### 修复
- 当预设映射或关联密钥字段变化后，刷新已激活预设导出的环境变量。
- 修复 FAQ 折叠组件在 render 阶段读取 ref 导致的 lint 问题。

## 0.5.4 - 2026-06-06

### 修复
- 修复 CI 环境下 DMG 打包失败的问题：移除 TAURI_BUNDLER_DMG_IGNORE_CI 环境变量

## 0.5.0 - 2026-06-04

### 新功能
- 密钥过期支持 — 为 API 密钥设置过期日期，在仪表盘上查看即将过期/已过期的密钥
- 应用内更新检查 — 在设置页面检查新版本并安装更新
- 仪表盘过期密钥提醒 — 高亮显示 7 天内过期或已过期的密钥
- 密钥元数据字段 — 为服务分组添加仪表盘 URL、文档 URL 和登录 URL
- 打开外部链接 — 点击 URL 在默认浏览器中打开

### 修复
- 移除密钥模型和预设中过时的 `modelName` 字段

### 性能优化
- 存储层重构，包含事务性审计日志记录和 N+1 查询修复

## 0.4.1 - 2026-06-03

### 修复
- 修复 TypeScript 构建错误：`DeletePresetButton` 中缺失的 prop 解构和未定义的错误处理函数

## 0.4.0 - 2026-06-03

### 新功能
- 新用户首次创建保险库时自动生成 "default" 预设
- 删除活跃预设时显示额外警告确认对话框

### 修复
- 删除预设后自动刷新列表并重新选中可用预设
- 删除活跃预设前先自动停用当前环境

### 性能优化
- 移除 Input 和 Textarea 组件中冗余的 `dir` 属性引用
